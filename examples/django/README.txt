Updating Django
===============

1. CherryPy
-----------

The project uses CherryPy for emulating a production django
server. Begin a simple Python web server, it allows to focus on
writing updates for the django framework.

CherryPy 3.8.0 should be downladed from
[here](https://pypi.python.org/pypi/CherryPy) and the cherrypy.diff
patch should be applied.
